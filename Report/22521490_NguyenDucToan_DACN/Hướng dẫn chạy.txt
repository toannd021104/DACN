Đặt file topology.json mô tả kiến trúc hệ thống vào thư terraform-generator

Chạy lệnh sau để tạo và áp dụng môi trường hạ tầng:
python3 generate.py [openstack|aws] N

Trong đó:
openstack hoặc aws: tên cloud provider
N: số lượng môi trường muốn tạo

Các thư viện cần thiết được hướng dẫn trên Readme của link GitHub.
